🔥 How to connect Firebase (5 minutes)

Step 1: Go to console.firebase.google.com and sign in with your Google account.

Step 2: Click "Add project" → name it HodgesHQ → disable Google Analytics → Create project.

Step 3: Click the web icon (</>) → register app as "HodgesHQ" → copy the firebaseConfig object shown.

Step 4: In the left menu go to Build → Realtime Database → Create database → choose your region → Start in test mode → Enable.

Step 5: Open this HTML file on a computer in a text editor → find the section marked PASTE YOUR FIREBASE CONFIG HERE → replace the placeholder with your config → save.

Step 6: On both phones — open the file in Chrome → tap the 3-dot menu → Add to Home Screen. Both phones now sync in real time.

📋 How standards work

Standards tab — tap each item to tick it off as the day progresses. The status badge shows "Standards met ✓" only when every item is ticked.

Mark today as done — once all standards are met, tap this button to record the day in the weekly tracker. The tracker auto-fills and shows the weekend screen unlock for that week.

Reset day — clears all ticks ready for the next day. The weekly tracker is not affected by resetting.

Weekly tracker resets automatically each Monday. The weekend screen unlock summary updates as days are marked.

📱 How gigs work

+ button records one occurrence. Progress fills toward the target. Tallies only awarded when the full target is hit.

− button removes the last tap if you tapped by mistake.

Instant gigs show a confirm dialog before awarding since there's no minus for these.

Missed days pause progress — they don't reset it. Keep tapping successful days.

Adding new gigs — come back to Claude and ask. Takes a few minutes, re-download the updated file.
